プロジェクト「発車メロディ再生アプリ」

このディレクトリは、YouTubeから発車メロディ動画の情報を収集し、
最終的にHTMLアプリとして利用するためのプロジェクトフォルダです。

== プロジェクトの再開方法 ==

このプロジェクトの作業を再開（新しい動画を100件追加収集）するには、
以下の手順を実行してください。

1. コマンドプロンプト（ターミナル）を開きます。

2. 以下のコマンドを実行します。
   python get_youtube_videos.py


== ファイル構成 ==

- get_youtube_videos.py:
  動画情報を収集するためのメインのPythonスクリプトです。

- keywords.txt:
  検索したいキーワードのリストです。このファイルを編集することで、
  収集対象の路線を自由に追加・変更できます。

- YouTube_Videos/ （このディレクトリは自動で作成されます）
  - train_melodies.csv: 収集した動画データ（表形式）。Excelなどで閲覧できます。
  - train_melodies.json: 収集した動画データ。HTMLアプリが利用します。
